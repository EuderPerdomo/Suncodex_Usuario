¿Como funciona este modulo?

Este modulo muestra la comparación entre el emplear un controaldor de tipo MPPT o un controlador de tipo PWM, partiendo de un estado de carga (SOC) y observando la curva de comportamiento para un día
Variables

  • Mapa: El mapa permite elegir una ubicación especifica pudiendo observar la diferencia de acuerdo a la radiación disponible.
  • SOC inicial: Este parametro permite modificar el % de carga del cual aprte la bateria, por defecto inicia en un 30%
  • Corriente de descarga: La corriente de descarga permite establecer un valor como descarga continua (representa un consumo de la bateria), por defecto el valor inicial es 0
  • Formulario Panel solar: Este formulario permite crear y editar los atributos para un panel solar personalizado
  • Selección de mes: Permite observar el comportamiento de carga para cada uno de los meses, para ello se toma la media de radiacion en cada mes y se calculan los valores de carga.
  • Grafico: Muestra la curva calculada de los estados de carga a lo largo de dia.
  • Porcentaje Final: Muestra el esatdo de carga final de las baterias para el controlador PWM / MPPT.
